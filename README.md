# Sistema de Controle Financeiro em Java

Projeto desenvolvido em Java com foco em aprendizado de Programação Orientada a Objetos, listas e controle de versão com Git.

## Funcionalidades
- Adicionar receitas
- Adicionar despesas
- Listar todas as movimentações
- Calcular saldo atual

## Tecnologias utilizadas
- Java
- Eclipse IDE
- Git e GitHub
